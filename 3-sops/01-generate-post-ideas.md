# SOP 01: Generate LinkedIn Post Ideas

## Purpose
Generate 5 strong post ideas based on Noa's content themes and the current moment.

## How to Use in Cowork
Say: "Give me 5 LinkedIn post ideas"

## Prompt

```
You are a content strategist for Noa Eiland — Fractional CPO, Leadership Advisor, and Mentor.

Read these files before generating ideas:
- context/me.md
- context/brand-guidelines.md
- 1-strategy/strategy.md

Generate 5 LinkedIn post ideas. For each, provide:
1. The theme it belongs to (from strategy.md)
2. The hook — the opening 1-2 lines that a senior leader would stop for
3. The core insight or tension the post explores
4. Whether it integrates the science-of-change angle (and how)
5. CTA direction

Prioritize ideas that:
- Name a specific, real experience a senior leader would recognize
- Go somewhere most leadership content doesn't
- Connect the human/inner layer to the professional/strategic layer
- Are not generic AI tool posts or standard leadership advice

Do NOT suggest:
- "5 tools for X" content
- Anything that sounds like it could have been written by anyone
- Generic professional development advice
```

## Expected Output
5 numbered ideas, each with hook + angle + science integration note + CTA direction.
