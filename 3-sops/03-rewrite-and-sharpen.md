# SOP 03: Rewrite and Sharpen a Draft Post

## Purpose
Take a draft Noa has written and make it tighter, sharper, and more on-brand — without losing her voice.

## How to Use in Cowork
Say: "Sharpen this post: [paste draft]"

## Prompt

```
You are editing a LinkedIn post for Noa Eiland.

Read these files first:
- context/brand-guidelines.md
- 1-strategy/strategy.md

DRAFT POST:
[PASTE DRAFT HERE]

Edit using these principles:

1. HOOK: Is the first line specific and true enough to stop a senior leader mid-scroll? If not, rewrite it.
2. BODY: Cut anything generic or vague. Every line should earn its place. Does it go somewhere?
3. SCIENCE LAYER: If there's an opportunity to ground the insight in neuroscience or behavioral research — add it (lightly, not academically).
4. VOICE CHECK: Remove corporate buzzwords, generic advice, exclamation marks, em dashes.
5. RHYTHM: One sentence per line. No walls of text.
6. CTA: Is it real and soft — a genuine question or invitation? Adjust if needed.

Return:
- The edited post, ready to publish
- A brief note (2-3 sentences) on the main changes and why
```

## Expected Output
Edited post + short rationale.
